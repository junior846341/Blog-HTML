Tags

(voltar)

Tag <i> Normalmente, é apresentado com o uso do tipo "itálico".
Tag <strong> é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas).
Tag <mark> representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.
Tag <hr> representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

Tags Texto

(voltar)

Tag <h1-h6> representam seis níveis de título de seção.
Tag <p> representa um parágrafo.
Tag <blockquote> indica que o texto incluído é uma longa citação.
Tag <small> representa comentários laterais e letras pequenas.
Tag <center> define o texto centralizado.

Listas Ordenadas

(voltar)

Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

Listas Não Ordenadas

(voltar)

Uma lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos). Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>.

Links e Atributos

(voltar)

Tag <a href=> cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.
Atributo target "_`blank" ele abre o link em uma nova aba.
Atributo title serve para adicionar um balão.
